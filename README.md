# CV Personnel Interactif

## Résumé du projet — CV Web

Ce projet est un **CV interactif en ligne** composé de 4 fichiers principaux : `index.html`, `style.css`, `jquery-scripts.js`, et `react-components.js`.

**Les langages utilisés** sont HTML5 pour la structure, CSS3 pour le design et les animations, et JavaScript pour l'interactivité.

**Les bibliothèques externes** sont jQuery 3.7.1 pour les interactions (accordion, tabs, scroll, typing effect), React 18 avec ReactDOM et Babel pour les composants dynamiques, Font Awesome pour les icônes, et Google Fonts (Poppins) pour la typographie.

**Le CV contient 7 sections** : une navigation fixe qui change de couleur au scroll, un header avec photo et effet de frappe animé, une section À propos avec des onglets, une section Compétences avec des barres de progression animées au scroll, une section Formation avec un accordion jQuery interactif, une section Projets avec des composants React ProjectCard incluant le vrai projet Gide.ma, et enfin une section Contact avec validation de formulaire.

**Les composants React créés** sont ProjectCard, TimelineItem, SkillBar et ContactInfo, chacun rendu dans un conteneur HTML dédié via ReactDOM.
