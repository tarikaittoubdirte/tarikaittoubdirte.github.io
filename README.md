# CV Personnel Interactif

Projet de Programmation Web 2 - Licence L2

## Technologies utilisées
- HTML5 / CSS3 (Flexbox, Grid, Responsive)
- jQuery (animations, validation, filtres, onglets, accordéon)
- ReactJS (4 composants fonctionnels avec props)
- GitHub Pages (hébergement)

## Structure du projet
- `index.html` : Structure sémantique du CV
- `css/style.css` : Mise en forme responsive
- `js/jquery-scripts.js` : Interactivité DOM
- `js/react-components.js` : Composants React (ProjectCard, TimelineItem, SkillBar, ContactInfo)
